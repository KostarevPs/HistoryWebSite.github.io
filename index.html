ВВЕДЕНИЕ В ЯЗЫКИ ПРОГРАММИРОВАНИЯ

Исторический контекст

Первый язык программирования высокого уровня, Plankalkül, был разработан Конрадом Цузе в 1940-х годах. FORTRAN, созданный в 1957 году, стал первым широко используемым языком.

Классификация

Языки программирования классифицируются по парадигмам: императивные, функциональные, логические, объектно-ориентированные. Каждая парадигма предлагает уникальный подход к решению задач.

Эволюция мышления

От машинного кода до языков высокого уровня с абстракциями. Современные языки фокусируются на читаемости, безопасности и параллелизме.

Тенденции развития

Современные тенденции включают рост популярности языков со статической типизацией, развитие domain-specific языков и увеличение важности систем обеспечения безопасности.

Java

Функция вычисления факториала

public class Factorial {
    public static int factorial(int n) {
        if (n == 0) {
            return 1;
        } else {
            return n * factorial(n - 1);
        }
    }
    
    public static void main(String[] args) {
        System.out.println(factorial(5));  // Вывод: 120
    }
}

Статическая типизация и обязательная структура класса

JavaScript

Функция вычисления факториала

function factorial(n) {
    if (n === 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

// Вызов функции
console.log(factorial(5));  // Вывод: 120

// Стрелочная функция (ES6+)
const factorialArrow = n => 
    n === 0 ? 1 : n * factorialArrow(n - 1);

Низкоуровневый контроль, ручное управление памятью

Rust

Функция вычисления факториала

fn factorial(n: u64) -> u64 {
    match n {
        0 => 1,
        _ => n * factorial(n - 1)
    }
}

fn main() {
    println!("{}", factorial(5));  // Вывод: 120
    
    // Итеративный вариант
    let factorial_iter = (1..=5).product::();
    println!("{}", factorial_iter);
}

Система владения, безопасность памяти без сборщика мусора

C#

Функция вычисления факториала

using System;

class Program {
    static int Factorial(int n) {
        return n == 0 ? 1 : n * Factorial(n - 1);
    }
    
    static void Main() {
        Console.WriteLine(Factorial(5));  // Вывод: 120
        
        // Современный синтаксис
        int FactorialLocal(int n) => 
            n == 0 ? 1 : n * FactorialLocal(n - 1);
    }
}

ЭВОЛЮЦИЯ ЯЗЫКОВ ПРОГРАММИРОВАНИЯ

Историческое развитие и ключевые инновации

1950-е

Эра мейнфреймов

FORTRAN (1957) - первый высокоуровневый язык, ориентированный на научные вычисления.

LISP (1958) - основа функционального программирования, обработка списков.

COBOL (1959) - для бизнес-приложений, читаемый английский синтаксис.

1970-е

Фундаментальные языки

C (1972) - низкоуровневый доступ с высокоуровневыми конструкциями.

SQL (1974) - декларативный язык для работы с базами данных.

Pascal (1970) - для обучения структурному программированию.

1990-е

Эра интернета

Python (1991) - акцент на читаемость и продуктивность.

Java (1995) - кроссплатформенность, виртуальная машина.

JavaScript (1995) - интерактивность веб-страниц.

Ruby (1995) - принцип наименьшего удивления, фреймворк Rails.

2000-е

Современная эра

C# (2000) - объектно-ориентированный язык Microsoft.

Scala (2004) - сочетание ООП и функционального программирования.

Go (2009) - от Google, упрощённый синтаксис, параллелизм.

Rust (2010) - безопасность памяти без сборщика мусора.

TypeScript (2012) - типизированное надмножество JavaScript.

2020-е

Эра специализации

Zig (2020) - современная замена C с безопасностью памяти.

Mojo (2022) - язык для AI, совместимый с Python.

Carbon (2022) - экспериментальный язык от Google, наследник C++.

Bun (2023) - быстрый JavaScript runtime и инструментарий.

ИНТЕРЕСНЫЕ ФАКТЫ И НАБЛЮДЕНИЯ

Малоизвестные детали о языках программирования

Первый баг

Термин "баг" (ошибка) появился в 1947 году, когда мотылёк застрял в реле компьютера Harvard Mark II. Техники приклеили его в журнал с пометкой "первый обнаруженный баг".

Происхождение Java

Изначально язык назывался Oak (Дуб), но из-за существующей торговой марки был переименован в Java. Название выбрано случайно во время перерыва на кофе.

Название Python

Язык назван не в честь змеи, а в честь британского комедийного шоу "Monty Python's Flying Circus", которое любил создатель Гвидо ван Россум.

JavaScript за 10 дней

JavaScript был создан Бренданом Айком всего за 10 дней в мае 1995 года для браузера Netscape Navigator. Первоначально назывался Mocha, затем LiveScript.

Код Аполлона

Программное обеспечение для миссии Аполлон было написано на языке ассемблера. Код содержал комментарии в стиле хайку для лучшей читаемости.

Язык для любви

Язык программирования "Whitespace" использует только пробелы, табуляции и переводы строк. Остальные символы игнорируются. Создан в 2003 году как шутка.

Первый программист

Ада Лавлейс (1815-1852) считается первым программистом в истории. Она написала первую программу для аналитической машины Чарльза Бэббиджа.

Swift и его логотип

Язык Swift от Apple назван в честь птицы стриж (swift). Логотип языка - стилизованная птица, символизирующая скорость и элегантность.
